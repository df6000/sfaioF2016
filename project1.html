Interactive Art Fall 2016

Project 1 -- Crit: 9/19

This first project will focus on performative interactivity. Maybe the first and simplest example of this in action is the Marcel Duchamp, "Fountain" piece that we talked about in class. The artist asks a question ("why is this here") and we are forced to answer it. Similar sorts of questions get asked in interface design. What do I do? What do I do next? What does this button do? What happens if...? These are the questions I want you to force people to ask themselves. My work in progress at www.thisisnotlaselva.net is an example of this stage of interactivity. A lot of the web is built on this model, as well as many musical instruments. Your project can be web based, or it can be another medium if you prefer. You can write code or you can use tools that other people have written, such as Flickr, Giphy, etc. Or if you prefer, you can present a mockup, with several sketches of an interface that you would like to build. Just, find an interesting way to engage the ideas that we're talking about in class!
