Interactive Art Fall 2016

Project 1 -- Crit: 9/19

In our second project we will build on the things we've already learned and add some new tricks to our arsenal that will allow us to create an interface/experience that is unique to every user and every moment -- in other words, a dynamic interface, that can change and evolve. Our goal will be to make a tool which people enjoy using, which feels like fun....in short, a game for making art. Like all of our other projects, this can be written in code but doesn't need to be.
